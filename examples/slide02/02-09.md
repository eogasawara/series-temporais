## Objetivo
Exibir uma serie composta por tendencia, sazonalidade, ciclo e ruido.

## Conexao com o PDF
Slide 09 do PDF `02-Estrutura-Temporal-em-Series-Temporais.pdf`, arquivo de saida `slides/02-09.png`.

## Descricao do codigo
Constroi tendencia, sazonalidade e ciclo, soma ruido e plota a serie composta.


``` r
source("https://raw.githubusercontent.com/eogasawara/series-temporais/main/code/utils.R")
set.seed(209)
```


``` r
# Parametros basicos
n <- 180
t <- 1:n
s <- 12

# Componentes
Tt <- 50 + 0.15 * t
St <- 4 * sin(2 * pi * t / s)
Ct <- 3 * sin(2 * pi * t / 60)
eps <- rnorm(n, 0, 1.2)

x <- Tt + St + Ct + eps
x_ts <- ts(x, frequency = s)

p <- forecast::autoplot(x_ts) +
  ggplot2::labs(x = NULL, y = NULL, title = "Interacao: tendencia + sazonalidade + ciclo + ruido")

out <- har_slide_file("02", "09")
har_ggsave_px(out, p)
```

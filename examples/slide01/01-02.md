## Objetivo
Gerar a figura do Slide 01 do PDF `01-Fundamentos-de-Series-Temporais-e-Processos-Estocasticos.pdf`.

## Conexao com o PDF
Slide 01, arquivo de saida `figures/01-01.png`.


``` r
source("https://raw.githubusercontent.com/eogasawara/series-temporais/main/code/utils.R")
set.seed(1)
```


``` r
# Serie com tendencia leve e ruido
T <- 120
t <- 1:T
x <- 0.03 * t + rnorm(T, sd = 0.8)

# Harbinger precisa de um objeto para plotagem
obj <- harbinger::harbinger()

har_save_plot(
  file = "figures/01-01.png",
  obj = obj,
  serie = x,
  idx = t,
  mark.cp = FALSE,
  width = TSED_WIDTH,
  height = TSED_HEIGHT
)
```

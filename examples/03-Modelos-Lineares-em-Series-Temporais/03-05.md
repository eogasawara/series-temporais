## Objetivo
Reproduzir a figura/resultado do slide 05 da Aula 03.

## Descricao
Este documento explica o passo a passo do codigo para gerar a figura/resultado associado ao slide.

## Conexao com o PDF
Slide 05 do PDF correspondente da Aula 03.

## Referencia
Hyndman, R.J., Athanasopoulos, G. (2021). Forecasting: Principles and Practice (3rd ed.). OTexts.


``` r
# Didatico: roteiro em 3 passos
# 1) Carregar dependencias/rotinas auxiliares.
# 2) Preparar dados e parametros do exemplo.
# 3) Gerar a figura/resultado esperado.
source("https://raw.githubusercontent.com/eogasawara/series-temporais/main/code/utils.R")

set.seed(3)
n <- 250

# AR(2): X_t = 0.6 X_{t-1} - 0.2 X_{t-2} + eps_t
x <- stats::arima.sim(model = list(ar = c(0.6, -0.2)), n = n)
x <- ts(x)

p <- forecast::autoplot(x) +
  ggplot2::labs(title = "SÃ©rie AR(2) simulada", x = "Tempo", y = "Xt")

f_out <- har_slide_file(3, 5)
har_ggsave_px(f_out, p, width_px = TSED_WIDTH, height_px = TSED_HEIGHT, dpi = TSED_DPI)
```
